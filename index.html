Projeto Principal

Dashboard Automatizado para Pequenos Negócios

Sistema web de BI que automatiza a entrega de dashboards para microempresas — sem depender de ferramentas pagas. O cliente envia seus dados (CSV), eu trato no Power Query e Python/Pandas, e o pipeline gera um dashboard interativo com gráficos, KPIs e insights automáticos, acessível pelo navegador.

  • Pipeline ETL completo: CSV → Python/Pandas → JSON → Dashboard web
  • Multi-tenant: cada cliente acessa seu painel por URL própria
  • Gráficos dinâmicos configurados via JSON (bar, line, doughnut, radar)
  • Insights automáticos gerados por regras de negócio por segmento
  • Seletor de mês com variação percentual sobre período anterior
  • Landing page com SEO, domínio próprio, robots.txt e sitemap

Dashboard Power BI — Análise de Vendas
