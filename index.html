Projeto 01 · Dashboard Automatizado

SaaS Dashboard para Pequenos Negócios

Sistema web de BI que automatiza a entrega de dashboards para microempresas sem ferramentas pagas. O cliente envia dados em CSV, o pipeline Python/Pandas trata e gera um painel interativo com gráficos, KPIs e insights automáticos — acessível por URL própria.

  • Pipeline ETL completo: CSV → Python/Pandas → JSON → Dashboard web
  • Multi-tenant: cada cliente acessa seu painel por URL própria
  • Gráficos dinâmicos configurados via JSON (bar, line, doughnut, radar)
  • Insights automáticos gerados por regras de negócio por segmento
  • Seletor de mês com variação percentual sobre período anterior

Projeto 02 · Call Center Analytics

MIS Dashboard — Análise Operacional de Call Center

Pipeline de MIS para operação de call center com 68 operadores (cobrança e atendimento). Lê planilha Excel com cabeçalho duplo mesclado, normaliza via Python/Pandas e gera dashboard dinâmico com ICM, TMA, NPS, CSAT, Absenteísmo — zero dado hardcoded.

  • ETL de xlsx com cabeçalho duplo mesclado via pandas (header=[0,1])
  • ICM — Índice Composto de Métricas calculado por operador
  • TMA médio, NPS, CSAT, DSAT, FCR, Rechamada e Transferência
  • Insights automáticos: top performers e alertas de absenteísmo
  • Dashboard atualiza com novo xlsx sem alterar nenhuma linha de HTML

Projeto 03 · Financial ETL

Análise Financeira com Pipeline ETL em Python

Pipeline end-to-end em Python/Jupyter: geração de dados transacionais sintéticos, extração e limpeza, modelagem de séries temporais e forecast de receita. Validação com SQL e exportação de relatórios automatizados.

  • Geração e validação de dados sintéticos (transações financeiras)
  • Limpeza e normalização com Pandas + detecção de outliers
  • Análise de séries temporais e forecast com modelos estatísticos
  • Schema SQL documentado com queries de análise gerencial

Projeto 04 · Vendas & Receita

Dashboard de Vendas — Análise Executiva

Dashboard analítico de vendas no Power BI com modelagem relacional em star schema, medidas DAX e layout para tomada de decisão executiva. Cobre faturamento, ticket médio, evolução mensal e metas por canal.

  • Modelagem relacional com star schema (fato + dimensões)
  • Medidas DAX: faturamento, ticket médio, variação MoM e YoY
  • Filtros e segmentações interativas por período, canal e produto
  • Metas vs realizado com indicadores visuais de performance
