## Sobre

## Projetos

### 📊 Dashboard Web para Pequenos Negócios
Sistema SaaS de BI completo com dados reais de clientes.

- **Pipeline ETL**: CSV → Python/Pandas → JSON → Dashboard interativo
- Multi-tenant: cada cliente acessa seu dashboard via URL própria
- 7+ gráficos dinâmicos por dashboard (bar, line, doughnut, radar)
- Insights automáticos gerados por regras de negócio customizadas por segmento
- Seletor de mês com variação percentual sobre período anterior

**Stack:** Python · Pandas · JavaScript · Chart.js · HTML/CSS · GitHub Pages

🔗 [Ver site](https://roniricardo.com) · [Ver código](https://github.com/rfrfrff3/roniricardo/tree/main/clientes)
