🧮 Fórmulas Essenciais — O arsenal completo 80/20

De centenas de funções do Excel, ~25 resolvem 80% do trabalho de análise de dados. Organizei em 4 níveis de prioridade. Domine os dois primeiros e você já está acima de 90% dos candidatos.

🟢 Nível 1 — Obrigatórias (aprenda estas primeiro)

PROCV / VLOOKUP — Buscar dados entre tabelas ▼

A fórmula mais pedida em entrevistas. Busca um valor na primeira coluna de uma tabela e retorna um valor da mesma linha em outra coluna.

// Buscar nome do vendedor pelo ID
=PROCV(A2; vendedores!A:B; 2; 0)

// Sintaxe:
=PROCV(valor_procurado; tabela; coluna_retorno; correspondência)
// correspondência: 0 = exata (SEMPRE use 0)
ABCD
1ID VendaVendedor_IDValorNome Vendedor
21001V03R$ 1.500=PROCV(B2;vendedores!A:B;2;0)
31002V01R$ 890→ "Ana Silva"

Limitação do PROCV: só busca para a DIREITA. Se o valor que você precisa retornar está à esquerda do valor buscado, o PROCV não funciona. Use ÍNDICE+CORRESP ou PROCX nesses casos.

PROCX / XLOOKUP — O substituto moderno do PROCV ▼

Disponível no Excel 365/2021+. Faz tudo que o PROCV faz e mais — busca para qualquer direção, sem precisar do número da coluna.

// Buscar nome pelo ID (qualquer direção)
=PROCX(B2; vendedores!A:A; vendedores!B:B; "Não encontrado")

// Sintaxe:
=PROCX(valor_procurado; matriz_pesquisa; matriz_retorno; [se_não_encontrado])

Na entrevista: se perguntarem PROCV, responda com PROCV. Mas mencione: "Também uso PROCX quando disponível, que é mais flexível — busca em qualquer direção e tem tratamento de erro nativo." Isso mostra que você está atualizado.

ÍNDICE + CORRESP — A dupla mais poderosa ▼

Combinação que supera o PROCV em tudo. Busca em qualquer direção, melhor performance, mais flexível.

// CORRESP encontra a POSIÇÃO do valor
=CORRESP(B2; vendedores!A:A; 0) → retorna 3 (linha)

// ÍNDICE retorna o valor na posição
=ÍNDICE(vendedores!B:B; 3) → retorna "Ana Silva"

// Combinados:
=ÍNDICE(vendedores!B:B; CORRESP(B2; vendedores!A:A; 0))

Quando usar cada um: PROCV → rápido e simples, busca à direita. ÍNDICE+CORRESP → busca em qualquer direção, melhor performance com muitos dados. PROCX → o melhor, mas exige Excel 365+.

SE / IF — Lógica condicional ▼

A base de toda lógica no Excel. Testa uma condição e retorna um valor ou outro.

// Classificar vendas
=SE(C2 >= 1000; "Alta"; "Baixa")

// SE aninhado (múltiplas faixas)
=SE(C2 >= 5000; "Premium";
  SE(C2 >= 1000; "Médio"; "Baixo"))

// SES (múltiplas condições sem aninhar — Excel 365+)
=SES(C2 >= 5000; "Premium";
     C2 >= 1000; "Médio";
     VERDADEIRO; "Baixo")
SOMASE / SOMASES — Somar com critério ▼

Soma valores que atendem a uma ou mais condições. Essencial para análise de dados.

// Somar vendas de SP
=SOMASE(B:B; "SP"; C:C)

// Somar vendas de SP no mês de Janeiro
=SOMASES(C:C; B:B; "SP"; D:D; "Jan")

// Somar valores acima de 1000
=SOMASE(C:C; ">1000"; C:C)

// Somar com referência de célula no critério
=SOMASE(B:B; F1; C:C)

CONT.SES / COUNTIFS — Contar com critério ▼
// Contar vendas de SP
=CONT.SE(B:B; "SP")

// Contar vendas de SP acima de R$ 100
=CONT.SES(B:B; "SP"; C:C; ">100")

// Contar valores únicos (fórmula clássica)
=SOMARPRODUTO(1/CONT.SE(A2:A100; A2:A100))
MÉDIASE / AVERAGEIF — Média condicional ▼
// Ticket médio de SP
=MÉDIASE(B:B; "SP"; C:C)

// Média de vendas acima de R$ 50
=MÉDIASES(C:C; C:C; ">50")

🟡 Nível 2 — Diferencial (te separa da multidão)

SOMARPRODUTO / SUMPRODUCT — Agregação poderosa ▼

Multiplica arrays e soma os resultados. Uma das fórmulas mais versáteis — substitui muitos SOMASES complexos.

// Faturamento total (quantidade × preço)
=SOMARPRODUTO(C2:C100 * D2:D100)

// Soma condicional com múltiplos critérios
=SOMARPRODUTO((A2:A100="SP") * (B2:B100="Corte") * C2:C100)

// Média ponderada (notas × pesos)
=SOMARPRODUTO(B2:B5 * C2:C5) / SOMA(C2:C5)

Poder oculto: SOMARPRODUTO pode fazer condicional sem ser SOMASES. Basta multiplicar critérios como (coluna="valor") — que vira 0 ou 1 — vezes o valor que quer somar. É tipo um filtro manual.

TEXTO / TEXT — Formatar datas e números como texto ▼
// Data como "Jan/2026"
=TEXTO(A2; "MMM/AAAA")

// Número como moeda
=TEXTO(B2; "R$ #.##0,00")

// Dia da semana
=TEXTO(A2; "DDDD") → "segunda-feira"

// Ano-Mês para ordenação
=TEXTO(A2; "AAAA-MM") → "2026-04"
Funções de TEXTO — ESQUERDA, DIREITA, EXT.TEXTO, CONCATENAR ▼

Essenciais para limpar e padronizar dados desestruturados.

// Extrair DDD de "(11) 99999-9999"
=EXT.TEXTO(A2; 2; 2) → "11"

// Primeiros 3 caracteres
=ESQUERDA(A2; 3)

// Últimos 4 caracteres (ex: extensão de arquivo)
=DIREITA(A2; 4)

// Juntar nome + sobrenome
=CONCAT(A2; " "; B2)
// Ou o moderno (Excel 365):
=A2 & " " & B2

// Limpar espaços extras
=ARRUMAR(A2)

// Padronizar maiúsculas
=MAIÚSCULA(A2) → "SÃO PAULO"
=PRI.MAIÚSCULA(A2) → "São Paulo"
=MINÚSCULA(A2) → "são paulo"

// Encontrar posição de um caractere
=LOCALIZAR("@"; A2) → posição do @

// Substituir texto
=SUBSTITUIR(A2; "São Paulo"; "SP")
Funções de DATA — ANO, MÊS, DIA, HOJE, DIATRABALHOTOTAL ▼
// Extrair componentes
=ANO(A2) → 2026
=MÊS(A2) → 4
=DIA(A2) → 1

// Data de hoje
=HOJE() → 01/04/2026

// Dias entre duas datas
=B2 - A2 → número de dias

// Dias úteis entre datas
=DIATRABALHOTOTAL(A2; B2)

// Diferença em meses
=DATADIF(A2; B2; "M") → meses completos

// Fim do mês
=FIMMÊS(A2; 0) → último dia do mês de A2

// Trimestre
="Q" & INT((MÊS(A2)-1)/3)+1 → "Q2"

Para BI: criar colunas de Ano, Mês, Trimestre, Dia da Semana é essencial antes de importar dados no Power BI. Facilita filtros e agrupamentos.

SEERRO / IFERROR — Tratamento de erros ▼
// PROCV que não dá erro se não encontrar
=SEERRO(PROCV(A2; tab; 2; 0); "Não encontrado")

// Divisão que não dá #DIV/0!
=SEERRO(A2/B2; 0)

// Sempre envolva PROCVs com SEERRO

Regra: todo PROCV/ÍNDICE+CORRESP deve estar dentro de um SEERRO. Planilha com #N/D aparecendo é sinal de amador. Em entrevistas, usar SEERRO automaticamente mostra profissionalismo.

🔴 Nível 3 — Avançado (impressiona na entrevista)

Fórmulas Matriciais e FILTRO / ÚNICO / CLASSIFICAR ▼

Funções de array dinâmico (Excel 365+). Revolucionaram o Excel — uma fórmula retorna múltiplos resultados.

// Filtrar vendas acima de R$ 1000
=FILTRO(A2:C100; C2:C100 > 1000)

// Lista de valores únicos
=ÚNICO(B2:B100)

// Classificar do maior para menor
=CLASSIFICAR(A2:B100; B2:B100; -1)

// Combinar: Top 5 vendedores únicos por faturamento
=CLASSIFICAR(ÚNICO(dados); ; -1)

Essas funções são o futuro do Excel. FILTRO, ÚNICO e CLASSIFICAR eliminam a necessidade de muitas macros e fórmulas complexas. Se sua empresa usa Excel 365, aprenda essas primeiro.

LET — Variáveis dentro de fórmulas ▼

Permite criar variáveis temporárias dentro de uma fórmula (Excel 365+). Melhora legibilidade e performance.

// Sem LET (confuso):
=SE(SOMASE(A:A;"SP";C:C)/SOMA(C:C)>0,3;"Forte";"Fraco")

// Com LET (legível):
=LET(
  totalSP; SOMASE(A:A; "SP"; C:C);
  totalGeral; SOMA(C:C);
  pct; totalSP / totalGeral;
  SE(pct > 0,3; "Forte"; "Fraco")
)
LAMBDA — Funções personalizadas sem VBA ▼

Crie suas próprias funções no Gerenciador de Nomes (Excel 365+). Reutilize lógica sem macro.

// No Gerenciador de Nomes, crie "CalcularImposto":
=LAMBDA(valor; aliquota; valor * aliquota)

// Usar na planilha:
=CalcularImposto(B2; 0,15) → B2 * 15%

// Converter Celsius para Fahrenheit:
=LAMBDA(c; c * 9/5 + 32)

LAMBDA é diferencial nuclear em entrevistas avançadas. Mostra que você sabe criar abstrações — mentalidade de programador dentro do Excel.

📊 Tabelas Dinâmicas — A feature #1 do Excel Crucial

Se você aprender UMA coisa de Excel, aprenda Tabelas Dinâmicas. É a funcionalidade mais poderosa, mais usada, e mais pedida em vagas. Uma Tabela Dinâmica faz em 30 segundos o que levaria horas com fórmulas. É essencialmente o GROUP BY + agregação do SQL, mas visual e interativo.

O que é e quando usar

Tabela Dinâmica (Pivot Table) pega uma tabela de dados brutos e permite resumir, agrupar, filtrar e analisar arrastando campos — sem fórmula nenhuma.

Como criar em 5 passos

  1. Selecione os dados — Clique em qualquer célula da tabela fonte (se estiver formatada como Tabela, melhor ainda)
  2. Inserir → Tabela Dinâmica — Atalho: Alt + N + V
  3. Escolha onde colocar — Nova planilha (recomendado) ou planilha existente
  4. Arraste campos — Linhas, Colunas, Valores e Filtros no painel à direita
  5. Configure agregações — Clique direito → Resumir por: Soma, Contagem, Média, etc.

Regra de ouro: SEMPRE formate seus dados como Tabela (Ctrl + T) antes de criar a Dinâmica. Isso garante que novos dados sejam incluídos automaticamente ao atualizar.

10 truques de Tabela Dinâmica que analistas usam

1. Agrupar datas por Mês/Trimestre/Ano ▼

Clique direito no campo de data → Agrupar → Selecione Meses, Trimestres, Anos.

Isso elimina a necessidade de criar colunas auxiliares de Mês ou Trimestre. A Dinâmica agrupa automaticamente. Funciona com qualquer campo de data.

2. Mostrar valores como % do total ▼

Clique direito no campo de valor → Mostrar Valores Como → % do Total Geral (ou % do Total da Coluna/Linha).

Outras opções úteis: % do Total da Linha (proporção por vendedor), Diferença de (comparar com mês anterior), % Diferença de (variação % MoM).

3. Campos Calculados ▼

Criar métricas derivadas dentro da Dinâmica sem alterar os dados fonte.

// Margem = Receita - Custo
Ferramentas de Tabela Dinâmica → Campos, Itens, Conjuntos → Campo Calculado

Nome: Margem
Fórmula: = Receita - Custo

// Ticket Médio = Receita / Qtd
Nome: Ticket_Medio
Fórmula: = Receita / Quantidade
4. Segmentação de Dados (Slicers) ▼

Filtros visuais bonitos que conectam múltiplas Dinâmicas.

Inserir → Segmentação de Dados → Selecione os campos (UF, Categoria, Ano).

Pro tip: conecte um Slicer a múltiplas Tabelas Dinâmicas → Clique direito no Slicer → Conexões de Relatório → marque todas as dinâmicas. Agora um filtro controla tudo — igual ao Power BI!

5. Linha do Tempo (Timeline) ▼

Inserir → Linha do Tempo → Selecione campo de data. Filtro visual por período (dia, mês, trimestre, ano) com slider.

6. Drill-down (Expandir/Recolher) ▼

Coloque múltiplos campos nas Linhas (ex: Região → UF → Cidade). Use os botões +/- para expandir/recolher hierarquias. Duplo clique expande uma linha específica.

7. Top N com Filtro de Valor ▼

Para mostrar apenas os Top 10 vendedores:

Clique na seta do campo Vendedor → Filtros de Valor → Primeiros 10 → Configure número e critério (ex: Top 10 itens por Soma de Faturamento).

8. Gráfico Dinâmico (Pivot Chart) ▼

Selecione a Tabela Dinâmica → Inserir → Gráfico Dinâmico. O gráfico se atualiza automaticamente com os filtros da dinâmica. Perfeito para apresentações rápidas.

9. Ordenar por Valor (não por nome) ▼

Clique direito na célula de valor → Classificar → Classificar do Maior para o Menor. Agora os vendedores/produtos ficam ordenados por faturamento, não por ordem alfabética.

10. Atualizar automaticamente ▼

Clique direito na Dinâmica → Opções de Tabela Dinâmica → Dados → Atualizar ao abrir o arquivo. Ou simplesmente: Alt + F5 para atualizar.

Cuidado: se adicionou novos dados, verifique se o intervalo da fonte expandiu. Se usou Ctrl + T (Tabela), isso é automático.

Se você já sabe SQL (e sabe!), Tabelas Dinâmicas são triviais. É a mesma lógica de GROUP BY + SUM/COUNT/AVG, mas com drag-and-drop. A vantagem é velocidade: 30 segundos vs. escrever e rodar uma query.

🔄 Power Query no Excel — Mesmo engine do Power BI ETL

Surpresa: o Power Query que você aprendeu no módulo de BI existe DENTRO do Excel. Mesma interface, mesma linguagem M, mesmas transformações. Se você sabe usar no Power BI, já sabe usar no Excel. Zero curva de aprendizado extra.

Como acessar

Dados → Obter Dados (ou Dados → Obter e Transformar em versões mais antigas)

Fontes que você pode conectar

  • Arquivos CSV / TXT
  • Outras planilhas Excel
  • Pasta inteira de arquivos (combinar múltiplos CSVs)
  • Bancos de dados (SQL Server, MySQL, Postgres)
  • APIs REST / Web
  • SharePoint, OneDrive
  • PDF (sim, extrai tabelas de PDF!)

Quando usar Power Query no Excel

  • Combinar 12 CSVs mensais em um
  • Limpar dados antes de analisar
  • Transformar relatórios do sistema (SAP, ERP)
  • Atualizar dados com um clique
  • Automatizar ETL sem VBA
  • Conectar a banco de dados direto

Caso prático: Combinar todos os CSVs de uma pasta

Cenário: você recebe 12 arquivos (jan.csv, fev.csv, ...) e precisa analisar tudo junto.

  1. Dados → Obter Dados → De Arquivo → De Pasta — Selecione a pasta
  2. Combinar e Transformar — O Power Query empilha todos os arquivos automaticamente
  3. Limpe — Remova colunas desnecessárias, corrija tipos, filtre linhas
  4. Fechar e Carregar — Dados vão para uma tabela no Excel
  5. Próximo mês? — Só jogue o novo CSV na pasta e clique Atualizar Tudo
// O que o Power Query gera nos bastidores (Linguagem M):
let
  Fonte = Folder.Files("C:\Dados\Vendas_2026"),
  Filtrado = Table.SelectRows(Fonte, each [Extension] = ".csv"),
  Combinado = Table.Combine(
    Table.TransformColumns(Filtrado, {
      {"Content", each Csv.Document(_, [Delimiter=";"])}
    })[Content]
  ),
  TiposCorrigidos = Table.TransformColumnTypes(Combinado, {
    {"data", type date}, {"valor", type number}
  })
in
  TiposCorrigidos

Isso é EXATAMENTE o que empresas precisam. "Temos 50 relatórios mensais em Excel, como juntamos?" → Power Query. Sem VBA, sem Python, sem complicação. E quando você migrar isso pro Power BI, é copiar e colar.

Regras de ouro para gráficos profissionais

❌ Erros de amador

  • 3D em gráficos (distorce proporções)
  • Pizza com 10+ fatias
  • Duas escalas Y sem identificar
  • Cores aleatórias sem padrão
  • Legenda longe do gráfico
  • Eixo Y que não começa em zero (barras)
  • Título genérico ("Gráfico 1")

✅ Boas práticas

  • Sempre 2D, limpo, sem firula
  • Pizza: máximo 5 fatias + "Outros"
  • Rótulos diretos nos dados (sem legenda separada)
  • Paleta de cores consistente (2-3 cores)
  • Título que conta uma história ("Vendas cresceram 15% no Q2")
  • Remover linhas de grade desnecessárias
  • Destacar o dado mais importante (cor diferente)

🧹 Limpeza de Dados — Onde 80% do tempo vai Prática

Verdade do mercado: analistas gastam ~60-80% do tempo limpando dados. O Excel é perfeito para limpezas rápidas antes de importar no Power BI ou antes de uma análise ad-hoc. Saiba esses truques e você economiza horas toda semana.

Checklist de limpeza — A sequência prática

  1. Remover duplicatas — Dados → Remover Duplicatas (selecione colunas-chave)
  2. Corrigir tipos — Números como texto? Selecione → bandeirinha verde → "Converter para Número"
  3. Tratar espaços — =ARRUMAR(A2) remove espaços extras e invisíveis
  4. Padronizar maiúsculas — =MAIÚSCULA(A2) ou =PRI.MAIÚSCULA(A2)
  5. Substituir valores — Ctrl + H (Localizar e Substituir)
  6. Tratar vazios — Selecione → Ctrl + G → Especial → Células em branco → Digite valor → Ctrl + Enter
  7. Texto para Colunas — Dados → Texto para Colunas (separar "São Paulo - SP" em duas colunas)
  8. Validar com CONT.SE — =CONT.SE(A:A;A2)>1 para achar duplicatas remanescentes
  9. Validação de dados — Dados → Validação (restringir entrada a listas, faixas numéricas)
  10. Contar antes × depois — Sempre registre: "tinha X linhas, após limpeza ficou Y"

Fórmulas de limpeza mais usadas

// Remover espaços e caracteres invisíveis
=ARRUMAR(TIRAR(A2))

// Extrair só números de texto misto ("ABC123")
// (Precisa de SUBSTITUIR encadeado ou Power Query — melhor PQ)

// Padronizar UF: pegar os 2 últimos caracteres
=MAIÚSCULA(DIREITA(ARRUMAR(A2); 2))

// Verificar se célula está vazia
=SE(A2=""; "VAZIO"; A2)

// Marcar duplicatas
=SE(CONT.SE($A$2:A2; A2) > 1; "DUPLICADO"; "OK")

// Converter texto de data para data real
=DATAVALOR(A2)

Melhor que limpar depois é evitar sujeira na origem. Use Dados → Validação de Dados:

🎯 Perguntas de entrevista sobre Excel

"Qual a diferença entre PROCV e ÍNDICE+CORRESP?" ▼

Resposta: "PROCV busca apenas para a direita e exige o número da coluna — se inserir colunas, a fórmula quebra. ÍNDICE+CORRESP busca em qualquer direção, é mais robusto com mudanças de estrutura e tem melhor performance com muitos dados. PROCX combina o melhor dos dois, mas exige Excel 365+."

"Como você analisa 50.000 linhas de vendas?" ▼

Resposta: "Primeiro formato como Tabela (Ctrl+T), crio uma Tabela Dinâmica para agrupar por período e categoria, uso Formatação Condicional para identificar outliers, e monto 2-3 gráficos para visualizar tendências. Se precisar de limpeza antes, uso Power Query. Em 15 minutos tenho uma visão completa."

"Já usou Power Query no Excel?" ▼

Resposta: "Sim, uso para combinar múltiplos arquivos de uma pasta, fazer limpeza e padronização de dados, e criar pipelines de transformação que se atualizam com um clique. É o mesmo engine do Power BI, então o conhecimento transfere diretamente."

"Referência absoluta vs relativa?" ▼

Resposta: "Relativa (A2) se ajusta ao copiar a fórmula — se copio para baixo, vira A3, A4... Absoluta ($A$2) fica fixa em qualquer direção. Mista ($A2 ou A$2) trava só coluna ou só linha. Uso F4 para alternar rapidamente."

$A2$A3, $A4...$A2 (coluna fixa)Tabelas cruzadas

"Como automatiza relatórios no Excel?" ▼

Resposta: "Uso Power Query para o ETL — ele memoriza todas as transformações. Os dados alimentam Tabelas Dinâmicas e gráficos que se atualizam automaticamente. Adiciono Slicers para interatividade. Quando os dados fonte mudam, basta clicar Atualizar Tudo (Ctrl+Alt+F5) e tudo se recalcula. Para entrega recorrente, configuro templates que o usuário final só precisa atualizar."
